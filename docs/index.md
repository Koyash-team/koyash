# KOYASH Documentation

KOYASH is a skincare product recommendation service. A user provides their
budget, ingredient constraints (vegan, cruelty-free, allergens to avoid), skin
concerns, and skin type, and the service assembles a structured "cosmetic bag"
of real products with a per-product justification.

This site hosts the project's **maintained documentation**. The source lives in
the [`docs/` folder of the repository](https://github.com/Koyash-team/koyash/tree/main/docs).
